# stock-tracker
A basic real-time stock tracking application. Built with the purpose of learning and understanding interesting system design concepts/technologies.


## To run

### Finhubb API key
Grab a free Finhubb API key from: https://finnhub.io/
Set the .env var FINHUBB_API_KEY to your API key.

### Running
In project root, run `docker compose up -d`

### Accessing
Navigate to `http://localhost:5173/` in a browser.
